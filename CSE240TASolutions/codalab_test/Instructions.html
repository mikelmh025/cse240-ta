Data

The datasets are downloadable from the Dataset page.

Code Submission

The participants must submit a zip file with their code via the Submission page.

Submission format

The final submission format is:

submission.zip

  • boostit.py
  • other python scripts you wrote

Note that to create a valid submission, zip all the file with 'zip -r zipfilename *' starting from this directory. DO NOT zip the directory itself, just its contents. THIS IS VERY IMPORTANT.
Here is a sample run.py file. The participants must submit a run.py with the exactly same format.

Computational restrictions

For both phases, the prescribed "time budget" is 300 seconds, including both the running time of your code and the evaluation time of the predicted results. In other words, the running time of your program must be less than 300 seconds.
